#컵라면 끓이는 법






1. 라면뚜껑을 완전 열고 스프를 넣는다
2. 뜨거운 물을 선까지 넣는다
3. 뚜껑을 덮고 3분 기다린다
4. 뚜껑을 열고 먹는다
